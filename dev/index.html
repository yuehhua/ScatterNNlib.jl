ScatterNNlib.scatter_add! — Function
scatter_add!(ys, us, xs)

Scatter addition operation.

ys[:, xs[k]...] = ys[:, xs[k]...] .+ us[:, k...]

Arguments

  • ys: the destination for us to aggregate to. This argument will be mutated.
  • us: the source data for aggregating.
  • xs: the mapping for aggregation from source (index) to destination (value).

The index of xs is corresponding to the index of us and the value of xs is corresponding to the index of ys. The value of xs can be Int or Tuple type.

The dimension of us must equal to one plus dimension of xs. ys, us and xs must be supported array type and be the same type.Array, StaticArray and CuArray are currently supported.
